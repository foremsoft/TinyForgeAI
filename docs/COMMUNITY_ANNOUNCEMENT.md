## Reddit (r/MachineLearning, r/Python, r/learnprogramming)

**Title:** [P] TinyForgeAI - Open source framework for training tiny LLMs - Looking for contributors!

```
Hey everyone!

I'm excited to share TinyForgeAI, an open-source framework we've been building for training small, focused language models from your own data.

**The Problem:**
Large LLMs are powerful but overkill (and expensive) for most tasks. Sometimes you just need a small model trained on your specific data.

**What TinyForgeAI Does:**
- Connects to your data sources (databases, Google Docs, Notion, PDFs, APIs)
- Trains tiny models using HuggingFace Transformers + LoRA
- Exports to ONNX and generates a complete FastAPI microservice
- Includes RAG support for document search

**Tech Stack:**
- Python 3.10+
- HuggingFace Transformers/PEFT
- FastAPI
- React dashboard
- Docker/Kubernetes ready

**We're Looking for Contributors!**

Whether you're a beginner or experienced, there's something for you:

- **Beginners:** We have "good first issue" labels and detailed contributing docs
- **Backend devs:** New connectors, training improvements
- **Frontend devs:** React dashboard enhancements
- **ML engineers:** Model optimization, new architectures
- **Tech writers:** Documentation and tutorials

**Links:**
- GitHub: https://github.com/foremsoft/TinyForgeAI
- Good First Issues: https://github.com/foremsoft/TinyForgeAI/labels/good%20first%20issue
- Contributing Guide: https://github.com/foremsoft/TinyForgeAI/blob/main/docs/CONTRIBUTING.md

Happy to answer any questions!
```

## GitHub Issue Labels to Create

For attracting contributors, create these labels:

| Label | Color | Description |
|-------|-------|-------------|
| `good first issue` | #7057ff | Good for newcomers |
| `help wanted` | #008672 | Extra attention is needed |
| `documentation` | #0075ca | Improvements to documentation |
| `enhancement` | #a2eeef | New feature or request |
| `bug` | #d73a4a | Something isn't working |
| `hacktoberfest` | #ff7518 | Hacktoberfest participation |
| `up-for-grabs` | #e99695 | Available for community contribution |
| `mentor available` | #7cfc00 | A maintainer will help with this issue |
